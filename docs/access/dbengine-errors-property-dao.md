# DBEngine.Errors Property (DAO)

Returns an **Errors** collection that contains all of the stored **Error** objects for the specified object. Read-only. 
  
## Syntax

 *expression*  . **Errors**
  
 *expression*  A variable that represents a **DBEngine** object.
