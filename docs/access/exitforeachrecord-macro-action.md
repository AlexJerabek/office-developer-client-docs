# ExitForEachRecord Macro Action

Use the **ExitForEachRecord** action to immediately exit a **[ForEachRecord](foreachrecord-data-block.md)** data block. 
  
> [!NOTE]
> The **ExitForEachRecord** action is available only in Data Macros.
