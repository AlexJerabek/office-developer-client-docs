# Recordset2.Fields Property (DAO)

Returns a **Fields** collection that represents all stored **Field** objects for the specified object. Read-only. 
  
## Syntax

 *expression*  . **Fields**
  
 *expression*  A variable that represents a **Recordset2** object.
