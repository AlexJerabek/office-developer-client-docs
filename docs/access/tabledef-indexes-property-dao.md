# TableDef.Indexes Property (DAO)

Returns an **Indexes** collection that contains all of the stored **Index** objects for the specified table. Read-only. 
  
## Syntax

 *expression*  . **Indexes**
  
 *expression*  A variable that represents a **TableDef** object.
